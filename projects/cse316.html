Overview

A full-stack web application developed as the final project for CSE 316 (Software Engineering) at Stony Brook University. Built with React.js frontend and Node.js backend, using MySQL for data persistence and Cloudinary for image storage.

This project demonstrates end-to-end web development skills including RESTful API design, database modeling, user authentication, and cloud-based media management.

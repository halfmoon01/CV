- CSS3
- JavaScript (ES6+)
- TailwindCSS

## 프로젝트 구조
```
/
├── index.html          # 메인 이력서 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # 인터랙션 스크립트
└── assets/
    └── images/         # 프로필 사진 등 이미지
```
